# practice-vault

**install**
```
npm install (package.json)
pip install -r (requirements.txt)
bundle install (Gemfile)
```
**run/test**
```
node <file>.js
python <file>.py
ruby <file>.rb
jasmine <file>.js
python -m pytest <file>.py
rspec <file>.rb
```
**update**
```
ncu -u (package.json)
pur -r (requirements.txt)
bundle update --all (Gemfile)
```
